Dota 2 page media

This folder belongs to the Dota 2 page only.

Use `src/assets/dota2/` to store images and GIFs used by the Dota 2 page.

Examples:
- `src/assets/dota2/heroes/lina.png`
- `src/assets/dota2/map/dota-map.jpg`
- `src/assets/dota2/animations/hero-intro.gif`

When updating the Dota 2 page, reference these files from the component/template using `assets/dota2/...` paths.
